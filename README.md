# Concrete-Strength-ML-SIT-FDP
Concrete compressive strength prediction using ML
Concrete Compressive Strength Dataset  
Source: [UCI Machine Learning Repository](https://www.kaggle.com/datasets/niteshyadav3103/concrete-compressive-strength)  

Features:
- Cement
- Slag
- Fly Ash
- Water
- Superplasticizer
- Coarse Aggregate
- Fine Aggregate
- Age

Target:
- Concrete Compressive Strength (MPa)
